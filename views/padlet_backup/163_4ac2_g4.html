    1. ปัญหาที่ทำให้ผู้เรียนต้องทำวิจัยเรื่องนี้ (รายบุคคล)
    1.1 ลิปสมัยใหม่ใส่สารเคมีเยอะขึ้นทำให้ปากคล้ำ จึงอยากทำลิปปลอดสารเคมีขึ้นเอง (เลขที่02)
    1.2 เกิดจากการที่เราไม่ค่อยดูแลรักษาริมฝีปากจึงทำให้มีสีปากไม่สวยและไม่มีความชุ่มชื้น (เลขที่03)
    1.3 ริมฝีปากแห้งลอกและดำทำให้ทาลิปบ่อย (เลขที่04)
    1.4  คนเราไม่รู้จักวิธีการดูแลรักษาริมฝีปาก จึงทำให้เกิดปัญหาต่างๆ เกี่ยวกับริมฝีปาก (เลขที่5)
    1.5 สาเหตุเดียวหรือหลายสาเหตุร่วมกันก็ได้หากดูแลไม่ดีพอหรือแก้ปัญหาไม่ตรงจุด ริมฝีปากยิ่งดูคล้ำลงไปมากกว่าเดิม (เลขที่10)

    2. อะไรคือสาเหตุของปัญหาตามข้อ 1 (รายบุคคล)

    2. อะไรคือสาเหตุของปัญหาตามข้อ 1 (รายบุคคล)
    2.1 ปัจจุบันมีคนใช้ลิปมากขึ้นและเลือกลิปที่ติดทนนานทำให้มีสารปนเปื้อนและสารเคมีจึงทำให้เกิดลิปบาล์มจากผลไม้นี้ขึ้นมา (เลขที่02)
    2.2 เกิดจากการเลียริมฝีปาก,เม้มปากรวมไปถึงการดื่มน้ำน้อยเกินไปเป็นเหตุสำคัญที่ทำให้ริมฝีปากแห้งเพราะร่างกายต้องได้รับน้ำเพื่อความชุ่มชื้นของร่างกาย (เลขที่03)
    2.3 ลิปสติกที่นำมาทาริมฝีปากอาจมีสารเคมีที่มีผลทำให้ริมฝีปากดำคล้ำและสภาพอากาศก็มีส่วนที่ทำให้ริมฝีปากแห้งลอก (เลขที่04)
    2.4 ปัจจุบันผู้คนใช้สาร หรือลิปสติกที่มีสารเคมี บางคนเกิดอาการแพ้ ทำให้ริมฝีปากเกิดการลอก และเป็นขุย (เลขที่05)
    2.5 ริมฝีปากดำคล้ำสาเหตุอีกประการหนึ่ง สาเหตุจากการเลียริมฝีปากบ่อย  (เลขที่10)

    3. นวัตกรรมอะไรที่ผู้เรียนจะใช้ในการแก้ปัญหานี้ (รายกลุ่ม)

    3. นวัตกรรมอะไรที่ผู้เรียนจะใช้ในการแก้ปัญหานี้ (รายกลุ่ม)
    3.1  การพัฒนาผลิตภัณฑ์ลิปบาล์ม ที่มีส่วนผสมของสารสกัดสีธรรมชาติ

    4. ผู้เรียนจะตั้งคำถามของการวิจัยไว้อย่างไรบ้าง (รายกลุ่ม)

    4. ผู้เรียนจะตั้งคำถามของการวิจัยไว้อย่างไรบ้าง (รายกลุ่ม)
    4.1 จะสามารถทำลิปบาล์มจากเปลือกผลไม้หรือผลไม้ได้หรือไม่ 
    4.2 สรรพคุณของ เปลือกแก้วมังกร,มะเขือเทศ,มะละกอสุก,มะนาว สามารถให้ความชุ่มชื้น ลดการหมองคล้ำ ช่วยให้ริมฝีปากอวบอิ่มอมชมพู ไม่แห้งและเป็นขุย แก่ริมฝีปากได้หรือไม่
    4.3 สรรพคุณของผลไม้ (เปลือกแก้วมังกร,มะเขือเทศ,มะละกอสุก,มะนาว) เมื่อนำมาใช้กับงานวิจัยสรรพคุณนั้นจะลดลงหรือไม่

    5. วัตถุประสงค์ของการวิจัย มีอะไรบ้าง (รายกลุ่ม)

    5. วัตถุประสงค์ของการวิจัย มีอะไรบ้าง (รายกลุ่ม)
    5.1 เพื่อศึกษาสรรพคุณของผลไม้ (เปลือกแก้วมังกร,มะเขือเทศ,มะละกอสุก,มะนาว)
    5.2 เพื่อศึกษาขั้นตอนและวิธีการทำลิปบาล์ม

    6. ผู้เรียนคาดเดาคำตอบที่เป็นสรุปผลของการวิจัยไว้อย่างไรบ้าง (รายกลุ่ม)

    6. ผู้เรียนคาดเดาคำตอบที่เป็นสรุปผลของการวิจัยไว้อย่างไรบ้าง (รายกลุ่ม)
    6.1 สามารถนำไปทาริมฝีปากได้
    6.2 ผลไม้ (เปลือกแก้วมังกร,มะเขือเทศ,มะละกอสุก,มะนาว) สามารถใช้สรรพคุณของผลไม้เพื่อทำผลิตภัณฑ์ได้

    11. ระบุตัวแปรที่ศึกษา (รายกลุ่ม)
    ขั้นที่  1 อบลมร้อนเปลือกแก้วมังกร,มะละกอ,มะเขือเทศ
    1) ตัวแปรต้น : เปลือกแก้วมังกร,มะละกอ,มะเขือเทศ
    2)  ตัวแปรตาม : เปลือกแก้วมังกร,มะละกอ,มะเขือเทศ อบแห้ง  2  ละลายขี้ผึ้งเชียร์บัตเตอร์
    1)  ตัวแปรต้น : ขี้ผึ้งและเชียร์บัตเตอร์
    2)  ตัวแปรตาม : การละลายของขี้ผึ้งและเชียร์บัตเตอร์
    ขั้นที่  3  ผสมขี้ผึ้งและเชียร์บัตเตอร์ที่ละลายแล้วกับผงเปลือกแก้วมังกร,มะละกอ,มะเขือเทศ
    1)  ตัวแปรต้น : ผงเปลือกแก้วมังกร,มะละกอ,มะเขือเทศ และ ขี้ผึ้ง, เชียร์บัตเตอร์ที่ละลายแล้ว
    2)  ตัวแปรตาม :   การผสมผงเปลือกแก้วมังกร,มะละกอ,มะเขือเทศ กับขี้ผึ้งและเชียร์บัตเตอร์เข้าด้วยกัน

    3.2 ของเดิมมีข้อด้อยหรือข้อที่เราต้องการแก้ไข/เปลี่ยนแปลง
         ของเดิมมีสรรพคุณผลไม้ที่ใช้ในงานวิจัยเพียงผลไม้ชนิดเดียว
         ดังนั้นเราจึงพัฒนาจากของเดิมที่ใช้ผลไม้เพียงชนิดเดียว โดยการเพิ่มผลไม้ เช่น มะละกอสุก, มะนาว, เปลือกแก้วมังกร, มะเขือเทศ ที่มีสรรพคุณบำรุงดูแล รักษา ริมฝีปากมาใช้ในงานวิจัยของเรา


    3.3 มีหลักการอะไรที่ใช้แก้ข้อด้อย หรือข้อที่ต้องการเปลี่ยนแปลง เราเลือกหลักการใด เพราะอะไร

    3.3 มีหลักการอะไรที่ใช้แก้ข้อด้อย หรือข้อที่ต้องการเปลี่ยนแปลง เราเลือกหลักการใด เพราะอะไร
    เลือกผลไม้ที่มีสรรพคุณ ที่สามารถบำรุงดูแล รักษาริมฝีปากได้ เพราะ การทำลิปบาล์มที่มีการบำรุงดูแล รักษาริมฝีปาก จึงต้องใช้ผลไม้ที่มีสรรพคุณนั้นได้


    3.4 จะประยุกต์หลักการที่เลือกเข้าสู่นวัตกรรมของเราได้อย่างไร

    3.4 จะประยุกต์หลักการที่เลือกเข้าสู่นวัตกรรมของเราได้อย่างไร
    การพัฒนาผลิตภัณฑ์ โดยการทำลิปบาล์มจากผลไม้ (มะเขือเทศ,มะละกอ,มะนาว,เปลือกแก้วมังกร) โดยคุณลักษณะของสีลิปบาล์ม เน้นใช้สารให้สรรพคุณของผลไม้เป็นหลัก

    บทคัดย่อ : การวิจัยครั้งนี้มีจุดมุ่งหมายเพื่อพัฒนาผลิตภัณฑ์ลิปบาล์จากผลไม้ (แก้วมังกร,มะละกอสุก,มะเขือเทศ) โดยศึกษาวิธีการอบผลไม้ด้วยเครื่องอบลมร้อน ที่อุณหภูมิ 60 องศาเซลเซียล เป็นเวลา 4 ชั่วโมง จากนั้นศึกษาอัตราส่วนของลิปบาล์มต่อผลไม้ที่ใช้ ต่อน้ำสะอาดที่เหมาะสม 3 สูตรสำหรับผลิตภัณฑ์ลิปบาล์ม แล้วนำไปตรวจสอบคุณภาพผลิตภัณฑ์เทียบกับเกณฑ์มาตรฐานผลิตภัณฑ์ชุมชนผู้ตรวจสอบ จำนวน 5 คน คัดเลือกสูตรอัตราส่วนที่เหมาะสมที่สุด ไปศึกษาการยอมรับของผู้บริโภคจากกลุ่มตัวอย่าง 30 คน โดยใช้แบบสอบถาม พบว่า อัตราส่วนที่เหมาะสมต่อการทำลิปบาล์ม คือ น้ำมันมะพร้าว : เชียร์บัตเตอร์ : ขี้ผึ้ง : อิลมัลชั่น : ผงแก้วมังกร(เปลือก) : ผงมะละกอ : น้ำมะนาว : น้ำสะอาด เท่ากับ 7.5 ml : 5 g : 5 g : 2.5 ml : 2 g : 2 g : 1.25 ml  : 15 ml และผู้บริโภคให้การยอมรับต่อผลิตภัณฑ์ลิปบาล์มจากผลไม้โดยรวม อยู่ในระดับมากที่สุด
